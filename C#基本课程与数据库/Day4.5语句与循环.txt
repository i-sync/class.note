==============================================================
			语句
代码(语句)块:由{开始,由}结束
   在代码块中声明的变量,对象,只能够用于该代码块

条件语句:就是用于执行条件分支,即根据条件表达式的值执行语句
        何时用:当需要根据不同条件进行判断,然后执行不同的操作时
	if(<条件表达式>)
	{
	    <语句块>
	}
   最简单条件语句if if...else
   多重if
   嵌套if
   switch
复杂条件语句
    多重if
    嵌套if
    switch

else匹配原则:else永远与离它最近的那个没有else的if匹配

else if 可以替换嵌套if

if与else后面如果只有一个语句,可以省略{}
若有多条语句,则必须使用{}

断点:当高亮显示为黄色时,说明程序执行到这句话了,
       但是语句还没有执行

查看变量的值:
1.看下面的局部变量(整体所有变量)
2.将鼠标放到局部变更上面(只想看单个变量)
3.选中你所要查看的数据,然后拖放到监视窗口中
         (你想看的不是单个变量)

switch注意事项:
1.两个case之间如果有语句,则必须写break;
2.case后的常量值不能相等.
3.default可以位于任意位置,不影响程序的执行顺序
   只有当所有case都未匹配上时才执行default
   并且default后也必须有break;
4.switch后的表达式类型只能是byte,short ,int ,long,char,string,bool
             object ,float,double不可以
5.一般用于三路(含)以上的选择

if嵌套适用于两/三路分支比较方便,而switch语句实现三路以上比较方便


===============================================================
			       循环
 循环:有条件控制性的反复执行一组指令
 语句的用途:
     对一组命令(语句块)执行一定的次数或反复执行直到指定的条件为真
     (取决于条件的Boolean值:运行的次数一般由特定变量来控制)
     1.反复执行一组相同指令
     2.反复执行一组相似指令
 三要素:
    1.循环变量必须要初始化
    2.在循环体中必须要修改循环变更的值
    3.必须能够保证循环变量有false(循环的结束条件)
 分析:
    1.先找循环变量(最终结果中变的那个量)
       再找初始值(最小值或者最大值)
    2.循环结束条件(循环变量与最小值或者最大值的关系)
       如果初始值为最小值,就用与最大值的关系
       如果初始值为最大值,就用与最小值的关系
    3.在循环体中改变循环变量(向着循环条件为false的方向改变)

  while     先判断后循环
  语法:
    while(表达式)
    {
      语句;        //有可能循环一次都不执行
    }
  
  do...while	先执行循环体中的语句,然后再判断条件是否为真
              如果为真则继续循环;如果为假,则终止循环;
  语法:
    do
    {
      语句;       //此循环将至少执行一次
    }
    while(表达式);

 逗号运算符举例:int i=5,j=6;

 逗号运算符的优先级在C#语言中最低


break：
  1.只可以用于循环中或者switch语句中,不能单独使用;
  2.用于终止循环,在多层循环中,中用于跳出一层循环
  3.一般与if一起使用,表明在某些条件下跳出循环

当不知道结束条件是什么时候,可以写while(true)但必须有break语句

contiune:
   1.只能用于循环语句中
   2.作用:跳过循环体中剩余语句,执行下次循环

嵌套循环:循环中还有循环
   执行过程:外循环一次,内循环N次(N次指内循环的总次数)


补充:  //调试程序
   1.当方法体的语句特别多,想调试加断点
   断点注意可以有多个,目的就是让程序在断点处暂停
   可按F11或F10进行调试
   当很多语句没有必要调试时,可设置行第二断点,直接按F5进入

   2.想用某个类
      1)知道类名
      2)看上面是否Using过了
      3)看项目的引用下面是否已经引用了
   
   3.项目的引用,常用三个组
   1..net:微软所提供给我们的最常用的命名空间
   2.项目:在做三层结构的时候,可以在项目中找到
          当前解决方案中的所有项目(.dll)
   3.浏览:第三方的类库(.dll)
   注意:引用里面的System.Drawing指的是类库,也就是*.dll
        using 里面的System.Drawing 指的是命名空间
	一般情况下类库名称与命名空间名称相同
