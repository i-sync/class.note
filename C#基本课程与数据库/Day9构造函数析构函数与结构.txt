							构造函数
---------------------------------------------------------------------------------
构造函数:
        与类同名,没有返回值类型,在创建对象时自动调用
构造函数的语法:
     [修饰符] 类名()
     {
		//构造函数的主体
     }

特点:
 1.构造函数的用途:通常用于初始 化(一般是为字段初始化)
   
 2.构造函数可以被重载.
  
 3. 每个类都必须至少有一个构造函数,如果你不提供
     编译器就自动生成一个没有参数的构造函数
     如果程序员自定义了构造函数,则编译器不再提供构造函数
     (所以建议:如果自定义构造函数,然后再显示声明默认构造函数)
 4.声明对象后,公共运行时操作如下
      1)分配内存
      2)调用构造函数做初始化操作
  5.构造函数一般声明成public,若声明为private ,则非本类就不能实例化了.

构造函数一般放在字段和属性之下,方法之上.

构造函数重载
       何时用:如果需要得到具有不同初始状态的对象,就可以用构造函数重载

初始化方式:
	1.构造函数
	2.Form_Load---------WinForm
	3.Page_Load---------Web

静态构造函数
     1.用于初始化类的静态成员
     2.可以和类的非静态构造函数共存
     3.不能有访问修饰符
     4.不能有任何参数
     5.只被执行一次,不管创建多少个对象,只在类的第一个对象创建之前执行

执行过程:
静态成员字段-->静态成员方法-->静态构造函数
成员字段-->成员方法-->构造函数


						析构函数
---------------------------------------------------------
析构函数是类中的特殊类型的方法:与类同名;没有返回值
 析构函数语法:
         ~类名()     //名称前加一个"~"符号
	 {
		//析构函数的主体
	 }
 析构函数由C#中的垃圾回收器调用 (没有访问修饰符,没有参数)
特点:
      1.在对象消毁时,自动调用的(CLR(垃圾回收器))
      2.作用为释放对象


 垃圾回收器是公共运行时的一个重要的组件,是用来释放内存的
     即:垃圾回收器定期或在内存状态要求[内存已满]时通过销毁
           不再需要或不再被引用的对象来释放内存
	   不再需要指:很长时间不操作的对象
	   不被引用指:对象==null

  垃圾回收器释放对象的条件(必须同时满足):
       1.对象不再被引用,也就是对象==null
       2.对象在销毁器列表中不被标记
           标记:创建对象时,CLR分配内存,同时,判断该类是否有析构函数
	   ,        有则在销毁器中标记,否则不在销毁器中标记
  垃圾回收器的工作原理:
         垃圾回收器定期或在内存状态要求时检查,发现没有被引用的对象
	 1.如果这个没有被引用对象在销毁器列表中没有被标识,就将这个对象立即清除
	 2.如果这个没有被引用的对象在销毁器列表中有标识
	    1)垃圾回收器就调用销毁器线程
	    2)销毁器线程调用这个对象的析构函数
	    3)这个对象的析构函数执行后,它在销毁器列表中的标记就被删除
	    4)等下次[或下下..次],垃圾回收器进行垃圾回收时就会将这个对象清除

   结论:在C#中,不要定义析构函数
    原因:1.就算不需要对象,也会在内存中驻留一定的时间
             2.销毁器需要独立的线程执行,并且该线程会占用大量的资源

							结构
-------------------------------------------------------------------
结构类型(Struct, Structure的缩写)
      1.以Struct关键字定义的数据
      2.可以有字段,方法和构造函数
      3.结构属于值类型,不能实现继承
 语法:
     Struct	<结构名>
     {
		//结构体
     }
 
 结构和类的区别
	0.结构是值类型,不能被继承
	   类是引用类型,可以被继承
	1.结构是值类型,保存在栈上
	2.结构用来定义轻量组的变量,即通常用来封装小型相关变量组
	3.结构成员变量的初始化: 
		结构不能在声明成员变量的地方初始化,必须在构造函数中初始化
	4.结构的默认构造函数由系统自动生成,不能自己声明;
		1)默认构造函数将所有成员变量都初始化为默认值
		2)如果期望结构成员变量拥有非默认的初始值,可以重载构造函数,
		   即定义有参数构造函数,同时必须把所有成员变量初始化
		(..当自定义有参构造函数后,编译器仍然提供一个无参构造函数
		..结构的构造函数必须将所有成员变量赋初值)
